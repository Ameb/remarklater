Link your reMarkable

Welcome to Remarklater, the best way to send any website to your Remarkable tablet. To get started, link Chrome to your tablet with a one-time code from Remarkable.

Once you've linked your account, you'll be able to ⌘P any page directly to your tablet.

Copy one-time code from Remarkable

Congrats, you're all set up!

Now let's give Remarklater a whirl by sending this page to your Remarkable tablet. Press ⌘P or go to the File menu and click Print…. In the printer dialog, select Send to Remarkable as your destination.

Once the file has printed, you should see it appear on your Remarkable tablet within a few moments. If you don't see the file show up, make sure your tablet is connected to WiFi. If you're sure your tablet is connected and are still running into issues, drop a line to michael@690labs.com.
