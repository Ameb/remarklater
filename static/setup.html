Link your reMarkable

Welcome to Send to reMarkable, the best way to send any page to your reMarkable tablet. To get started, link Chrome to your tablet with a one-time code from reMarkable.

Once you've linked your account, you'll be able to ⌘P any page directly to your tablet.

Copy one-time code from reMarkable

Congrats, you're all set up!

Now let's give Send to reMarkable a whirl by sending this page to your reMarkable tablet. Press ⌘P or go to the File menu and click Print…. In the printer dialog, select Send to reMarkable as your destination.

Once the file has printed, you should see it appear on your reMarkable tablet within a few moments. If you don't see the file show up, make sure your tablet is connected to WiFi. If you're sure your tablet is connected and are still running into issues, drop a line to m@mschade.me.
